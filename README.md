# Medical Insurance Cost Prediction

## Project Overview
This project uses Linear Regression to predict medical insurance charges based on individual health and demographic factors. It was developed as part of an ICE Activity to practice Exploratory Data Analysis (EDA) and Machine Learning.

## Dataset
The dataset includes features such as:
* **Age, Sex, BMI, Children, Smoker Status, and Region.**
* **Target Variable:** `charges` (Medical insurance cost).

## Key Findings
* **Smoker Status** is the most significant predictor of high insurance costs.
* The model achieved an **R² score of approximately 0.78**, meaning it explains 78% of the variance in the data.

## Technologies Used
* Python (Pandas, NumPy, Seaborn, Matplotlib)
* Scikit-Learn (Linear Regression)
* Jupyter Notebook
